# Support Tone Guidelines - Neutral & Solution-Focused

**Principle:** Never assume or comment on user emotions. Stay neutral, professional, and solution-focused.

---

## Core Rules

### ‚úÖ DO

1. **Focus on Solutions**
   ```
   ‚úÖ "Let me help you get this working."
   ‚úÖ "Here's how to fix that."
   ‚úÖ "I'd be happy to help with that!"
   ```

2. **Acknowledge the Issue Neutrally**
   ```
   ‚úÖ "Let me help you with adding todos."
   ‚úÖ "I can explain how that works."
   ‚úÖ "Here's what to do for [issue]."
   ```

3. **Provide Clear Steps**
   ```
   ‚úÖ "To fix this:

       2. Go to Privacy
       3. Enable Microphone for helpem"

   ```
   ‚úÖ "If that doesn't solve it, our support team at support@helpem.ai can help directly."
   ‚úÖ "For [billing issue], please contact support@helpem.ai. They'll assist you right away."

   ```
   ‚úÖ "Is there anything else I can help with?"
   ‚úÖ "Let me know if you have other questions!"

### ‚ùå DON'T

1. **Never Assume Emotions**
   ```
   ‚ùå "I understand you're frustrated."
   ‚ùå "That must be annoying."
   ‚ùå "I'm sorry you're upset."
   ‚ùå "Don't worry!"
   ‚ùå "I know this is difficult."

   ```
   ‚ùå "You seem confused..."
   ‚ùå "Looks like you're having trouble..."
   ‚ùå "I can tell you're angry..."
   ```

3. **Never Be Condescending**
   ```
   ‚ùå "Obviously, you need to..."
   ‚ùå "Simply just..."
   ‚ùå "All you have to do is..."
   ‚ùå "It's easy, just..."

   ```
   ‚ùå "You need to enable permissions."
   ‚ùå "You should have checked settings first."
   ‚ùå "You forgot to..."
   ```

5. **Never Guess or Be Uncertain**
   ```
   ‚ùå "I think it might be..."
   ‚ùå "Maybe try..."
   ‚ùå "It could be that..."
   
   ‚úÖ ESCALATE instead: "Let me connect you with our support team for accurate help."

## Example Conversations

**Problem:** Assumes frustration, comments on emotions, overly apologetic

### ‚úÖ GOOD - Neutral & Solution-Focused
