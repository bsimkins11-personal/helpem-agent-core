## Email Setup Checklist

### DNS Configuration

**MX Records (for receiving):**
```
Priority 1: mx1.forwardemail.net
Priority 5: mx2.forwardemail.net
```

**SPF Record:**
```
v=spf1 include:spf.forwardemail.net -all
```

**DKIM Record:**
```
(Generated by email provider)
```

**DMARC Record:**
```
v=DMARC1; p=quarantine; rua=mailto:dmarc@helpem.ai
```

## Support Team Workflow

### When Email Arrives

1. **Auto-reply sent** (SLA acknowledgment)
2. **Check AI support logs** (did user already try AI?)
3. **Categorize ticket:**
   - Billing ‚Üí Handle immediately
   - Bug ‚Üí Reproduce and escalate to dev
   - How-to ‚Üí Answer + update AI knowledge base
   - Feature request ‚Üí Log for product roadmap
4. **Respond within SLA**
5. **Log improvement** (if AI should have handled it)

### Post-Response

1. **Update AI knowledge base** (if new issue pattern)
2. **Add to FAQ** (if common question)
3. **Track resolution time**
4. **Ask for feedback** (optional)
