**Symptoms:**
- Average message cost > $0.0006
- System prompt growing beyond 5K tokens
- Context memory causing high costs

**Mitigation:**
- Monitor token usage per message
- Optimize system prompt quarterly
- Use summary instead of full history (Premium memory)

- Single user sending 5,000+ messages/month
- API calls for automation (scripts)

**Mitigation:**
- Fair use policy (3,000 msgs/month)
- Rate limiting (50 msgs/hour)
- Friendly email if abuse detected (not hard block)

#### 4. Founder Time Diverted to Billing Infrastructure

**Symptoms:**
- Building custom subscription backend
- Debugging receipt validation
- Handling refund requests manually

**Mitigation:**
- **Use RevenueCat** (already planned) ‚úÖ
- Apple handles refunds automatically
